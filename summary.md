# FUCKME/FUCKYOU - Art Installation Application

## Overview
FUCKME/FUCKYOU is an interactive art installation that creates a confrontational yet reflective experience by having participants hear their own voice deliver harsh self-criticism. The application combines voice cloning technology with AI-generated content to create a digital mirror that reflects exaggerated self-criticism back to the user.

## Core Functionality
1. **Voice Recording**: Captures a 10-second audio sample of the user's voice
2. **Image Capture**: Takes a photo of the user via webcam
3. **AI-Generated Critique**: Uses Gemini AI to generate a harsh, exaggerated self-critique based on the user's image
4. **Voice Cloning**: Processes the voice sample through F5-TTS to clone the user's voice
5. **Spoken Feedback**: Delivers the AI-generated critique in the user's own cloned voice

## Technical Implementation
- **Frontend**: Minimalist HTML/CSS/JavaScript interface with a step-by-step guided experience
- **Backend**: Node.js server that handles API requests and file processing
- **AI Integration**: Connects to Google's Gemini API for generating personalized critiques
- **Voice Technology**: Utilizes ComfyUI with F5-TTS for high-quality voice cloning
- **Multilingual Support**: Available in both English and German

## Artistic Context
The installation creates a unique psychological experience where participants confront their own image through the lens of their harshest inner critic, spoken in their own voice. This creates a powerful moment of reflection on self-perception, self-criticism, and the relationship between our appearance and identity. The provocative title "FUCKME/FUCKYOU" encapsulates the duality of self-love and self-criticism that many people experience.
